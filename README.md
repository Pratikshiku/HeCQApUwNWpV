### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、活动管理员管理、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告
5. 组团活动
6. 个人中心
7. 组团活动收藏
8. 组团活动报名
9. 后台管理

**活动管理员管理功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告管理
5. 组团活动管理
6. 组团活动留言管理
7. 组团活动报名管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（班级管理、学院管理、专业管理、公告类型管理、组团活动类型管理）
4. 公告管理
5. 组团活动管理、组团活动留言管理、组团活动收藏管理、组团活动报名管理
6. 用户管理
7. 活动管理员管理
8. 轮播图信息
